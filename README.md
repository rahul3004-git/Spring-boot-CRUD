# CRUD Web Application

This project is a CRUD-based application built with **Spring Boot**.  
It demonstrates basic Create, Read, Update, and Delete operations with MySQL database integration.

---

## Features
- **Create** – Add new records to the database.  
- **Read** – View and retrieve existing records.  
- **Update** – Modify existing records.  
- **Delete** – Remove records from the database.  
- **MySQL Integration** – Store and manage data efficiently.  
- **RESTful API Endpoints** – Access data via HTTP requests.

---

## Tech Stack
- **Java**  
- **Spring Boot**  
- **Spring Data JPA**  
- **MySQL**  
- **Maven**
